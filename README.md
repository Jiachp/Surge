# Surge 运行状态面板

用于 Surge iOS 的状态面板。

## 功能

- 显示 Surge 运行时间
- 显示 MITM 状态
- 显示 Rewrite 状态
- 显示 Scripting 状态
- 点击面板重新加载配置

## Surge 配置

```ini
[Script]
Surge 运行状态 = type=generic, script-path=你的Raw地址, argument=icon=speedometer&color=#F6C970
